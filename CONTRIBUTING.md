# Contributing to Heapi

First off, thank you for considering contributing to Heapi! It's people like you that make Heapi such a great tool.

## Code of Conduct

By participating in this project, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

### Reporting Bugs

- Use a clear and descriptive title for the issue.
- Describe the exact steps which reproduce the problem in as many details as possible.
- Explain which behavior you expected to see instead and why.

### Suggesting Enhancements

- Use a clear and descriptive title.
- Provide a step-by-step description of the suggested enhancement.
- Explain why this enhancement would be useful to most Heapi users.

### Pull Requests

- Pull Requests are always welcome!
- Ensure your code follows the Go and TypeScript coding standards used in the project.
- Update documentation as necessary.
- Document any new features or changes in `CHANGELOG.md` (if exists).

## Development Setup

1. Install [Go](https://golang.org/dl/)
2. Install [Node.js](https://nodejs.org/)
3. Install [Wails](https://wails.io/docs/gettingstarted/installation)
4. Clone the repo: `git clone https://github.com/your-username/heapi.git`
5. Run in dev mode: `wails dev`
